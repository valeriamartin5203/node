# chat con amigos

## Descripcion

En este proyecto se realizo un chat que puedas hablar con mas personas en vivo como en los juegos en linea, este proyecto se creo para la clase de programacion para internet

## Tecnologias utilizadas
Para este proyecto se uso:
Html
Css
Node.js
Express.js
Socket.io
JavaScript

## Caracteristica
Las caracteristicas de este proyecto es:
comunicacion en tiempo real
soporte a muchos usuarios
Intefaz sencilla
